# Getting Started

To start using the CheckVIES API, follow these steps:

1. **Obtain an API Key**: Sign up at [checkvies.com](https://checkvies.com) to receive your API key.
2. **Review the API Reference**: Check out our [API Reference](/docs/api/checkvies-automated-api) to understand the available endpoints.
3. **Integrate**: Use our [client examples](/docs/clients) to quickly integrate CheckVIES into your application.

## Authentication
CheckVIES uses API keys to authenticate requests. You can find your API key in your account dashboard.
Include your API key in the `X-Api-Key` header of your requests.
