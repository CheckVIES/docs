# Introduction

Welcome to the CheckVIES documentation. CheckVIES is a high-performance service designed to validate VAT numbers against the VIES (VAT Information Exchange System) database.

Our service provides:
- Real-time VAT validation.
- Batch processing capabilities.
- Detailed error reporting.
- Reliable API with high availability.

Get started by following our [Getting Started](./getting-started.md) guide.
